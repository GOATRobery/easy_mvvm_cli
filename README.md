# EasyMvvm

The cli for the EasyMVVM framework, which help you quick start with a template project.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'easy_mvvm'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install easy_mvvm
